IDEA 1

ชื่อโปรเจกต์ (ตุ๊กตา): "MarketWise - Smart Crypto & Stock Analyzer"
1. Features (ฟีเจอร์หลัก)
แบ่งเป็น 2 ส่วนคือ Crypto (สายเทคนิคกราฟ) และ Stocks (สายพื้นฐาน Valudation)

A. โหมด Crypto (Technical Analysis)
Real-time Price: แสดงราคาเหรียญหลัก (BTC, ETH, ADA, etc.)
Trend Status (Highlight Feature): บอกสถานะปัจจุบันทันที
🟢 Bullish (ขาขึ้น): เมื่อ RSI < 30 (Oversold) หรือ MACD Signal ตัดขึ้น
🔴 Bearish (ขาลง): เมื่อ RSI > 70 (Overbought) หรือ MACD Signal ตัดลง
⚪ Neutral (พักตัว): ค่าอยู่กลางๆ
Indicator Details: แสดงค่าตัวเลขที่คำนวณได้จริง เช่น RSI: 72.5, MACD: 0.0024 ให้ผู้ใช้เห็น
B. โหมด หุ้น (Fundamental Analysis)
Stock Search: ค้นหาชื่อหุ้น (เช่น AAPL, TSLA, PTT)
Financial Health:
โชว์ค่า P/E Ratio (Price-to-Earnings).
โชว์ค่า EPS (Earnings Per Share).
Valuation Verdict: ระบบช่วยวิเคราะห์ความถูกแพง
🟢 Undervalued (น่าเก็บ): P/E ต่ำกว่ากณฑ์ (เช่น < 15)
🔴 Overvalued (แพงไป): P/E สูงเกินเกณฑ์ (เช่น > 30)
⚠️ Loss Warning: แจ้งเตือนถ้าบริษัทขาดทุน (EPS ติดลบ)
C. ฟีเจอร์ทั่วไป
Refresh System: ปัดลงเพื่อรีเฟรชข้อมูล (SwipeRefreshLayout)
Watchlist: บันทึกเหรียญหรือหุ้นที่ชอบไว้หน้าแรก
2. Design & UI/UX (แนวคิดการออกแบบ)
เนื่องจากเป็นแอปข้อมูล ตัวเลขจะเยอะ เราต้องเน้น "อ่านง่าย และ สีสื่อความหมาย"

Color Palette (ธีมสี):
Background: Dark Mode (สีดำเทา #121212) เหมาะกับแอปเก็งกำไร ดูแล้วโปร
Text: สีขาว หรือ เทาอ่อน
Positive (ขึ้น/ถูก/ซื้อ): สีเขียวสว่าง (#4CAF50)
Negative (ลง/แพง/ขาย): สีแดง (#F44336)
Neutral: สีเหลือง หรือ เทา
Layout Structure (โครงสร้างหน้าจอ):
Home Screen: แสดงรายการ Watchlist เป็น Card เรียงลงมา แสดงชื่อ + ราคาปัจจุบัน + ป้าย Status เล็กๆ (Bull/Bear).
Detail Screen (เมื่อกดเข้าไป):
Header: ชื่อเหรียญ/หุ้น โลโก้ขนาดใหญ่
Price Section: ราคาตัวใหญ่ๆ
Analysis Card (หัวใจสำคัญ): กรอบสี่เหลี่ยมเด่นๆ
ข้อความสรุป: "STRONG BUY" หรือ "WAIT"
แถบสี (Gauge View): เหมือนหน้าปัดความเร็ว ชี้ว่าอยู่โซนเขียวหรือแดง
Data Table: ตารางแสดงค่า P/E, RSI, High, Low
Chart: กราฟเส้นง่ายๆ (ใช้ Library MPAndroidChart)
3. Technical Plan (แผนการพัฒนา)
ลำดับการทำเพื่อให้ไม่สับสน แบ่งเป็น 3 Phase:

Phase 1: The Fetcher (ดึงข้อมูลพื้นฐาน)
Goal: ดึงราคาปัจจุบันมาโชว์ให้ได้ก่อน
API แนะนำ:
Crypto: CoinGecko API (ฟรี, ไม่ต้องใช้ Key สำหรับ Basic tier, ใช้ง่ายมาก)
Stock: Alpha Vantage หรือ Finnhub (ฟรีจำกัดจำนวนครั้ง, ต้องสมัคร Key)
Task:
ออกแบบ Data Model (Class Coin, Stock).
Setup Retrofit เรียก API ราคาปัจจุบัน (/simple/price ของ CoinGecko).
ทำ RecyclerView โชว์รายการราคา.
Phase 2: The Logic (ใส่สมองให้แอป)
Goal: ดึงข้อมูลย้อนหลัง และคำนวณตัวเลข
API:
Crypto: เรียก endpoint market_chart (CoinGecko) เพื่อขอราคาปิดย้อนหลัง 30 วัน
Task:
สร้าง Class TechAnalysisHelper ใน Kotlin:
ฟังก์ชัน calculateRSI(prices: List<Double>): Double
ฟังก์ชัน calculatePE(price: Double, eps: Double): Double
นำข้อมูล JSON ย้อนหลังที่ได้ เข้าฟังก์ชันนี้
เอาผลลัพธ์มาแสดงผล เปลี่ยนสี Text ตามเงื่อนไข (High RSI = สีแดง เป็นต้น)
Phase 3: The Polish (ตกแต่งและกราฟ)
Goal: ทำกราฟและ UI ให้สวย
Task:
ติดตั้ง Library MPAndroidChart.
นำข้อมูล List ราคาปิด มาพลอตกราฟเส้น
ใส่รูปลูกศรขึ้นลง อนิเมชันเล็กน้อยตอนโหลดข้อมูล
